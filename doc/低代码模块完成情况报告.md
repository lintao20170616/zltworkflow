## 3. 未完成 / 缺失功能

### 3.1 画布区域（核心缺失）

| 缺失项 | 说明 |
|--------|------|
| drop 事件处理 | 无法接收从组件库拖入的组件 |
| dragover 事件处理 | 无拖入时的视觉反馈 |
| 组件渲染 | 中间区域仅为 `el-empty`，无组件树渲染 |
| 组件选择 | 无法通过点击选中组件 |

**影响**: 无法完成「拖拽 → 落画布 → 选中 → 配置」的闭环。

### 3.2 组件库与 Schema 不一致

| 问题 | 说明 |
|------|------|
| 硬编码 | ComponentLibrary 组件列表未复用 componentSchemas |
| 分类不全 | 仅有 form，缺少 basic、layout、data 等 |
| 组件不全 | componentSchemas 有 20+ 组件，库中仅 12 个 |

### 3.3 画布操作

| 缺失项 | 说明 |
|--------|------|
| 删除组件 | 无 Delete 键或删除按钮 |
| 复制/粘贴 | store 有 duplicateComponent，无 UI 入口 |
| 拖拽排序 | 无画布内组件排序 |
| 层级调整 | 无上移/下移 |
| 右键菜单 | 无 |

### 3.4 预览与持久化

| 缺失项 | 说明 |
|--------|------|
| 实时预览 | 无 PC/移动端预览 |
| 后端 API | 仅 localStorage，无服务端持久化 |
