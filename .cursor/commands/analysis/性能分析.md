# 性能分析

分析项目的性能特征和优化机会：

## 1. 响应性能分析
- 关键接口响应时间
- 数据库查询性能
- 前端渲染效率

## 2. 资源使用评估
- CPU和内存使用情况
- 并发处理能力
- 资源瓶颈识别

## 3. 数据访问性能
- SQL查询优化机会
- 索引使用情况
- 缓存策略效果

## 4. 算法和数据结构
- 算法时间复杂度
- 数据结构选择合理性
- 循环和递归优化

## 5. 性能优化建议
- 关键瓶颈点
- 具体优化方案
- 实施优先级

## 输出要求：
**保存到 `.cursor/性能优化报告.md`，包含：**
- 性能基准和瓶颈分析
- 优化方案和预期收益
- 实施计划和验证方案
